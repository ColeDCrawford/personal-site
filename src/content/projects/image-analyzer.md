<p>Final project for CSCIE90 - Cloud Computing. This is an application which allows users to analyze images using AWS Rekognition to provide object label predictions and text recognition. The application is fully serverless, using Vue for the frontend and various AWS services (S3, API Gateway, DynamoDB, Rekognition, CloudFront, ACM, and Lambda) for the backend.</p>
