CatchPy is the HarvardX annotation storage backend — part of the AnnotationsX stack — implementing the W3C Web Annotation Data Model over Django / Postgres with JWT auth, and maintaining backward-compatibility with the older AnnotatorJS API. I contributed and deployed it as an annotation layer across several DARTH projects that needed persistent, per-user annotations on text and IIIF images.
